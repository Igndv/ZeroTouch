# UI Design: Launcher (Glassmorphism)

## Theme Consistency
Launcher uses the same [Glassmorphism] tokens as the HUD for a unified experience.

## Layout
- **Container**: 400x350 Frameless window.
- **Header**: Large glowing title "ZeroTouch".
- **Body**: Translucent controls.
- **Footer**: High-contrast "START" button.

## Interactive States
- **Hover**: Cyan glow expansion.
- **Pressed**: Subtle scale down (98%).
- **Checked**: Neon highlight.
